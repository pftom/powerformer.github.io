Mar 20, 2017 | 想法 | Hits

我现在的技术方向是

接下来的想法大致是

  • 深入运用Github
  • 熟悉Markdown
  • 逐渐加深对hexo等模板建站的认识，不断改进本博客
  • 把整个前端自己学到的和没学的选一个方向深入并系统的学习，并写一个系列的技术栈学习心得
  • 数据库驱动ORM
  • 后端同上
  • 学习关于搜索引擎的知识，以及web优化等
  • 把一整套：网站响应式,移动端，小程序，微信开发都去尝试和了解
  • 尝试去阅读和理解前端和后端的一些著名库的设计思想，并写下自己的理解
  • fork下他们的代码，并把自己觉得不好的地方 pull request
  • 在上面的基础上自己尝试用现有的模式+设计思想编写相应的库，模块，模板引擎，优化方案
  • 自己尝试去写一些好用的编辑器
  • 去玩黑客
  • 了解关于操作系统的知识
  • 温习Linux，体会Unix设计哲学
  • 接触算法，加速对算法的认识
  • 考研
